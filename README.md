# English Mix — План разработки

## Текущее состояние

Сейчас есть прототип веб-приложения, который:
- Показывает заранее подготовленные тексты с английскими вставками
- Позволяет нажать на слово для показа перевода
- Имеет кнопки "Показать перевод всех слов" и "Сбросить"

## План дальнейшей разработки

Пункты расположены в порядке выполнения. Каждый пункт — законченная фича, которую можно протестировать независимо.

---

### 1. Скачать словарь частотности английских слов

**Что сделать:** Найти и скачать датасет с частотами английских слов (word frequency list). Подойдут:
- [Google Books Ngram](https://storage.googleapis.com/books/ngrams/books/datasetsv2.html) — но он слишком большой
- [SUBTLEX-US](https://www.ugent.be/pp/experimentele-psychologie/en/research/documents/subtlexus) — частоты на основе субтитров
- [Word Frequency Data](https://www.wordfrequency.info/) — бесплатные списки
- Готовый opensource-датасет типа [english-word-frequency](https://github.com/hermitdave/FrequencyWords) на GitHub

**Формат хранения:** JSON-файл `data/word_frequencies.json` вида:
```json
{
  "the": 52846529,
  "of": 30992462,
  "and": 26812431,
  ...
}
```

**Проверка:** Скрипт `python scripts/check_frequencies.py` загружает файл и выводит топ-10 слов с частотами.

### 2. Написать скрипт для разбора FB2

**Что сделать:** Создать Python-скрипт `scripts/parse_fb2.py`, который:
- Принимает путь к FB2-файлу
- Извлекает текст книги
- Разбивает на главы (по тегам `<section>` или `<title>` в FB2)
- Сохраняет результат в JSON вида:
```json
{
  "title": "Название книги",
  "chapters": [
    { "title": "Глава 1", "text": "текст главы..." },
    ...
  ]
}
```

**Проверка:** `python scripts/parse_fb2.py sample.book.fb2` создаёт `output/book.json` с корректной структурой.

---

### 3. Написать скрипт для подбора слов для замены

**Что сделать:** Создать скрипт `scripts/select_words.py`, который:
- Принимает фрагмент текста (главу), словарь частотностей и параметры (уровень сложности A1-C2, процент замены)
- Находит в тексте английские слова, которые есть в словаре
- Фильтрует по уровню сложности (A1 = самые частые, C2 = редкие)
- Сортирует кандидаты по частоте встречаемости в тексте (чтобы одни и те же слова повторялись)
- Выбирает слова для замены, пока не наберётся нужный процент от текста
- Возвращает список пар {en, ru} для замены

**Уровни сложности (ориентировочно):**
| Уровень | Диапазон частот |
|---------|----------------|
| A1      | топ-1000       |
| A2      | 1001–3000      |
| B1      | 3001–8000      |
| B2      | 8001–15000     |
| C1      | 15001–30000    |
| C2      | > 30000        |

**Проверка:** `python scripts/select_words.py --input chapter.json --freq word_freq.json --level B1 --percent 8` выводит список замен и статистику.

### 10. Интеграция полного цикла: FB2 → готовый текст в приложении

**Что сделать:** Финальный скрипт `scripts/process_book.py`, который объединяет п.2–4:
- Принимает FB2-файл и параметры (уровень, процент замены)
- Разбирает, подбирает слова, генерирует JSON
- Сохраняет результат в папку `data/books/`
- Веб-приложение загружает книги из этой папки

**Проверка:** `python scripts/process_book.py --input book.fb2 --level B1 --percent 8` → в `data/books/` появляется JSON, который подхватывается веб-приложением.
